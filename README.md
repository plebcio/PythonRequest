# The bestest RPC project ever

## Requirements
* Linux
* Cmake 3.15 or higher
* A python3 instalation under /usr/bin
* A c++20 compatible compiler
* (possibly more)

## How to build
0. Go to the top dir
1. `mkdir build`
2. `cd build`
3. `cmake .. `
4. `make ` 

## Running
1. Navigate to the top directory 
2. Run `./build/bin/main`
